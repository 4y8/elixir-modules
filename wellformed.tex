\section{Definitions}

We define (non-empty) paths, module names and extended names:

$$P ::= X \;|\; P.X $$
$$M ::= P \;|\; P[\overline{\tx = t}]$$
$$\Next ::= N \;| \; \tx$$

Note that modules names are flattened.
Module $P.X$ is no longer a child of $P$; they are two distinct, incomparable paths. \\

We define module declaration and redefine type environments:

$$\mathfrak{M} ::= \Moddecl \;|\; (\overline{\tx : \star}) \rightarrow \Moddecl $$
$$\Gamma ::= \overline{P : \mathfrak{M}}, \overline{D}$$

The environment contains a module environment followed by local declarations (both can be empty).
For the environment to be well formed, the declarations $D$ should never have the form $X : T$.
This will be formalized in the corresponding section.

We also define:

$$\local(\overline{P : \mathfrak{M}}, \overline{D}) = \overline{D}
    \quad\text{ and }\quad
    \modules(\overline{P : \mathfrak{M}}, \overline{D}) = \overline{P : \mathfrak{M}}$$


\subsection{ident}

We define $\ident :\forall \alpha \beta.\; \alpha : \beta \rightarrow \alpha$ by:

$$\ident(\alpha : \beta) = \alpha$$

\subsection{find}

We define find: $\forall \alpha \beta.\; \overline{\alpha : \beta} \times \alpha \rightharpoonup \alpha : \beta$,
where $\rightharpoonup$ means find is only partially defined.

\[
    \begin{prooftree}
        \infer0{ \find{\gamma, \alpha : \beta}(\alpha) = \alpha : \beta }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \hypo{ \find{\gamma}(\alpha) = \alpha : \beta }
        \hypo{ \alpha' \neq \alpha }
        \infer2{ \find{\gamma, \alpha' : \beta'}(\alpha) = \alpha : \beta }
    \end{prooftree}
\]

If the derivation rules above are not applicable to the inputs, then find is not defined.

We also define:
\[
    \begin{prooftree}
        \hypo{ \find{\Sigma}(\BX) = \BX : \BXdecl }
        \infer1{ \find{\Sigma}(\BX[\overline{\tx = t}]) = \overline{D_{[\overline{t/x}]}} }
    \end{prooftree}
\]

and
\[
    \find{\Gamma}(\Next) = \find{\local(\Gamma)}(\Next) \qquad
    \find{\Gamma}(P) = \find{\modules(\Gamma)}(P)
\]

\subsection{dom}

We define $\dom: \forall \alpha \beta.\; \overline{\alpha : \beta} \rightarrow \Parts(\alpha)$ by:

\[
    \dom(\epsilon) = \emptyset
    \qquad
    \dom(\gamma, \alpha : \beta) = \dom(\gamma) \;\cup\; \{ \alpha \}
\] \\

We also define: $\dom(\Gamma) = \dom(\modules(\Gamma)) \cup \dom(\local(\Gamma))$.

Note that $ \dom(\modules(\Gamma))$ and $\dom(\local(\Gamma))$ are disjoint.

\subsection{decls}

\[
    \decls(\BX : \BXdecl) = \{\overline{\tx}\} \cup \left(\bigcup\overline{\dom(D)} \right)
\]

We define possibly empty paths:
$$\pi ::= \epsilon \;|\; P $$

We define $\flatten, \merge$ and $\withargs$ by:

$$
    \left\{\begin{array}{ll}
        \flatten(P : \Moddecl[])                                        & = P : \Moddecl[]                                                                       \\
        \flatten\left(P : \Moddecl[X : T, \overline{D}]\right)          & = \flatten\left(P : \Moddecl\right), \flatten(P.X : T)                                 \\
        \flatten\left(P : \Moddecl[D, \overline{D'}]\right)             & = \merge\left(P : \Moddecl[D], \flatten\left(P : \Moddecl[\overline{D'}]\right)\right) \\
        \flatten\left(P : (\overline{\tx : \star}) \rightarrow T\right) & = \withargs_{\overline{\tx}} \left( \flatten(P: T) \right)                             \\
    \end{array}\right.
$$

$$
    \begin{array}{ll}
        \withargs_{\overline{\tx}}\left(P : \Moddecl, \overline{P' : \mathfrak{M}}\right) = \left( P : (\overline{\tx : \star}) \rightarrow \Moddecl\right), \overline{P' : \mathfrak{M}} \\ \\
        \merge\left(P : \Moddecl[D'], P : \Moddecl, \overline{P' : \mathfrak{M}}\right) = P : \Moddecl[D', \overline{D}], \overline{P' : \mathfrak{M}}
    \end{array}
$$

We define $\Pushenv$ (TODO use a better syntax) by:

$$
    \left\{\begin{array}{lll}
        \Pushenv[\Gamma][\pi][X : T]                                                                 & = \Gamma, \flatten(\pi.X : T)                       \\
        \Pushenv[\Gamma, P : \{ \overline{D'} \}] \text{ if } \ident(D) \notin \dom(\overline{D'}) & = \Gamma, P : \{ \overline{D'}, D \}              \\
        \Pushenv[\Gamma, P' : \{ \overline{D'} \}] \text{ if } P' \neq P                           & = \left(\Pushenv\right), P' : \{ \overline{D'} \} \\
        \Pushenv[\Gamma, D']                                                                       & = \left(\Pushenv\right), D'                       \\
        \Pushenv                                                                                   & \text{undefined otherwise}
    \end{array}\right.
$$

We will denote $\vdash \Pushenv$ whenever $\Pushenv$ is well defined.
This is roughly equivalent to saying that $\Gamma$ does not contain a module $P$ that already exports $\ident(D)$.
Note that $\Pushenv$ being defined does not necessarily imply that it is well formed. \\

We can always declare a new module:

\[
    \begin{prooftree}
        \hypo{ \pi; \Sigma \vdashgam \Gamma}
        \infer1{ \pi; \Sigma\vdashgam \modules(\Gamma), \pi.X : \{\}, \local(\Gamma)}
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \hypo{ \pi; \Sigma \vdashgam \Gamma}
        \infer1{ \pi.X; \Sigma\vdashgam \modules(\Gamma), \pi.X : \{\}}
    \end{prooftree}
\]

We can also always inject local declarations without exporting them:

\[
    \begin{prooftree}
        \hypo{ \pi; \Sigma \vdashgam \Gamma}
        \infer1{ \pi; \Sigma\vdashgam \Gamma, D}
    \end{prooftree}
\]

This rule will be used to check wellformedness of arrow types and parametric modules,
when it is necessary to temporarily add the parameters in the environment.

It is important to note that exported modules will never be added to the local environment.
Although the environment would be technically wellformed, a declaration that tries to refer
to an exported module using a relative path would not - as formalized below.

\paragraph{Wellformedness rules:}

We will denote $P; \Sigma; \Gamma \vdashtyp \overline{T}$ to say that all types in $\overline{T}$
are well formed. Note that $P; \Sigma; \Gamma \vdashdecl \overline{D}$ will NOT mean the same, as
formalized in the next section.

We define: $\makeenv$:

\[
    \makeenv : \left\{\begin{array}{ll}
        (\Gamma, P : \{ \tx : \star \})              & \mapsto \left(\Pushenv[\Gamma][P][\tx : \star]\right), \tx : \star                           \\
        (\Gamma, P : \{ \tx : [= t] \})              & \mapsto \left(\Pushenv[\Gamma][P][\tx : [= t]]\right), \tx : [= t]                           \\
        (\Gamma, P : \{ x : \bigcap \overline{T} \}) & \mapsto \left(\Pushenv[\Gamma][P][x : \bigcap \overline{T}]\right), x : \bigcap \overline{T} \\
        (\Gamma, \pi : \{ X : \mathfrak{M} \})       & \mapsto  \left(\Pushenv[\modules(\Gamma)][\pi.X][\mathfrak{M}]\right), \local(\Gamma)
    \end{array}\right.
\]

\[
    \begin{prooftree}
        \infer0{ \epsilon; \Sigma\vdashgam \epsilon }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \hypo { P; \Sigma \vdashgam \Gamma }
        \hypo { \vdash \Pushenv[\Gamma][P][\tx : \star] }
        \infer2 { P; \Sigma \vdashgam \makeenv(\Gamma, P : \{ \tx : \star \}) }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \hypo { P; \Sigma \vdashgam \Gamma }
        \hypo { \vdash \Pushenv[\Gamma][P][\tx : [= t]] }
        \hypo { P; \Sigma; \Gamma \vdashtyp t }
        \infer3 { P; \Sigma \vdashgam \makeenv(\Gamma, P : \{ \tx : [= t] \}) }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \hypo { P; \Sigma \vdashgam \Gamma }
        \hypo { \vdash \Pushenv[\Gamma][P][x : \bigcap \overline{T}] }
        \hypo { P; \Sigma; \Gamma \vdashtyp \overline{T} }
        \infer3 { P; \Sigma \vdashgam \makeenv(\Gamma, P : \{ x : \bigcap \overline{T} \}) }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \hypo { \pi; \Sigma \vdashgam \Gamma }
        \hypo { \pi.X; \Sigma; \modules(\Gamma), \pi.X : \{ \} \vdashtyp \mathfrak{M} }
        \infer2 { \pi; \Sigma \vdashgam\makeenv(\Gamma, \pi : \{ X : T \})  }
    \end{prooftree}
\]

\section{Declaration lists}

For the sake of brevity, all type and behavior environments used on the left of $\vdash$ are implicitely assumed to be well-formed
in this section and the next section.

We extend the definition of $\makeenv$ with:

$$\makeenv(\Gamma, (\overline{\pi : \{ D \}}, \pi' : \{ D' \})) = \makeenv(\makeenv(\Gamma, \overline{\pi : \{ D \}}), \pi' : \{ D' \})$$

\[
    \begin{prooftree}
        \infer0{ \pi; \Sigma; \Gamma \vdashtyp \epsilon }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \hypo{ \pi; \Sigma; \Gamma \vdashdecl \overline{D} }
        \hypo{ \pi; \Sigma; \makeenv(\Gamma, \overline{\pi : \{D\}}) \vdashdecl D' }
        \infer2{ \Sigma; \Gamma \vdashdecl \overline{D}, D' }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \hypo { \pi; \Sigma \vdashgam \makeenv(\Gamma, \pi : \{D\}) }
        \infer1 { \pi; \Sigma; \Gamma \vdashdecl D  }
    \end{prooftree}
\]

\subsection{Module types}

\[
    \begin{prooftree}
        \infer0{ \pi;\Sigma; \Gamma \vdashbxinst \epsilon }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        %\hypo{ \Sigma; \Gamma \vdash \BX : \Lambda \left( \overline{\tx : \star} \right) \{ \overline D \} }
        \hypo { \pi;\Sigma; \Gamma \vdashtyp \BX [\overline{\tx = t}]  }
        \infer1{ \pi; \Sigma; \Gamma \vdashbxinst \BX [\overline{\tx = t}]  }
    \end{prooftree}
\]
\\
We disallow implementation of conflicting behaviors:

\[
    \begin{prooftree}
        \hypo { \pi;\Sigma; \Gamma \vdashbxinst \overline{\BX[\overline{\tx = t}]}}
        \hypo { \pi;\Sigma; \Gamma \vdashbxinst \BX' [\overline{\tx = t}] }
        \hypo { \left(\bigcup \overline{\decls(\find{\Sigma}(\BX))}\right) \cap \decls(\find{\Sigma}(\BX')) = \emptyset }
        \infer3{ \pi; \Sigma; \Gamma \vdashbxinst \overline{\BX [\overline{\tx = t}]}, \BX'[\overline{\tx = t'}]  }
    \end{prooftree}
\]
\\

\[
    \begin{prooftree}
        \hypo{  \pi;\Sigma; \Gamma \vdashdecl \overline D }
        \hypo{  \pi;\Sigma; \Gamma \vdashbxinst \overline {\BX [\overline{\tx = t}]} }
        \hypo{\left(\bigcup \overline{\find{\Sigma}(\BX[\overline{\tx = t}])}\right) \subset \overline{D}}
        \infer3{  \pi;\Sigma; \Gamma \vdashtyp \Moddecl}
    \end{prooftree}
\]

Here we consider unordered set inclusion, although $\overline{D}$ is an ordered set.

\[
    \begin{prooftree}
        \hypo{ \vdashlamb \Lambda (\overline{\tx : \star})  }
        \hypo{  \pi;\Sigma; \Gamma, \overline{\tx : \star} \vdashtyp T }
        \infer2{  \pi;\Sigma; \Gamma \vdashtyp (\overline{\tx : \star}) \rightarrow T }
    \end{prooftree}
\]

\[

\]
