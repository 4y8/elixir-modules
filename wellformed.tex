\section{Some functions}

All behavior and type environments that appear in this section are assumed to be well formed.

We define extended names:
$$\Next ::= N \;| \; \tx$$

\subsection{ident}

We define $\ident: \left\{ \begin{array}{c}
        \Beh \rightarrow \BX \\
        D \rightarrow \Next
    \end{array}  \right.$ by:

$$\ident(\BX : \BXdecl) = \BX$$

\[
    \ident(\tx : \star) = \tx
    \qquad
    \ident(\tx : [= t]) = \tx
\]

\[
    \ident(X : T) = X
    \qquad
    \ident(x: \bigcap \overline{T}) = x
\]

\subsection{find}

We define find: $\left\{ \begin{array}{c}
        \Sigma \times \BX \rightharpoonup \Beh \\
        \Gamma \times \Next \rightharpoonup D
    \end{array}  \right.$
($\rightharpoonup$ means find is only partially defined) by:

\[
    \begin{prooftree}
        \infer0{ \find{\Sigma, \BX : \BXdecl}{\BX} = \BX : \BXdecl }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \hypo{ \find{\Sigma}{\BX} = \Beh }
        \hypo{ \ident(\Beh') \neq \BX }
        \infer2{ \find{\Sigma, \Beh'}{\BX} = \Beh }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \infer0{ \find{\Gamma, X : T}{X} = X : T }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \infer0{ \find{\Gamma, x : \bigcap \overline{T}}{x} =  x : \bigcap \overline{T} }
    \end{prooftree}
\]
\[
    \begin{prooftree}
        \infer0{ \find{\Gamma, \tx : \star}{\tx} = \tx : \star }
    \end{prooftree}
    \qquad
    \begin{prooftree}
        \infer0{ \find{\Gamma, \tx : [= t]}{\tx} = \tx : [= t] }
    \end{prooftree}
\]

\[
    \begin{prooftree}
        \hypo{ \find{\Gamma}{\Next} = T }
        \hypo{ \ident(D) \neq \Next }
        \infer2{ \find{\Gamma, D}{\Next} = T }
    \end{prooftree}
\]
\\

We also define:
\[
    \begin{prooftree}
        \hypo{ \find{\Sigma}{\BX} = \BX : \BXdecl }
        \infer1{ \find{\Sigma}{\BX[\overline{\tx = t}]} = \overline{D_{[\overline{t/x}]}} }
    \end{prooftree}
\]

Note that find is a partial function.
If the derivation rules above are not applicable to the inputs, then find is not defined.

\subsection{dom}

We define $\dom: \left\{ \begin{array}{c}
        \Sigma \rightarrow \Parts(\BX)   \\
        \Gamma \rightarrow \Parts(\Next) \\
        \Beh \rightarrow \Parts(\Next)
    \end{array}  \right.$ by:

\[
    \dom(\epsilon) = \emptyset
\]
\[
    \dom(\Sigma, \Beh) = \dom(\Sigma) \;\cup\; \{ \ident(\Beh) \}
    \qquad
    \dom(\Gamma, D) = \dom(\Gamma) \;\cup\; \{ \ident(D) \}
\]

\[
    \dom(\BX : \BXdecl) = \left( \bigcup \overline{\{\tx\}} \right) \cup \left(\bigcup\overline{\dom(D)} \right)
\]
NOTE: or just $\left(\bigcup\overline{\dom(D)} \right)$?

behavior environment of size one.

\subsection{destruct}

$\text{destruct} : D \rightharpoonup \Parts(D)$ will allow us to destruct module paths later:

$$
    \text{destruct} : \left\{ \begin{array}{ll}
        X : \Moddecl                                      & \mapsto \{ \overline{D} \} \\
        X : (\overline{\tx : \star}) \rightarrow \Moddecl & \mapsto \{\overline{D}\}   \\
        \_                                                & \mapsto \text{undefined}
    \end{array} \right.
$$
